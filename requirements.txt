# ── ML — Class imbalance handling (NEW) ───────────────────
# imbalanced-learn provides SMOTE (Synthetic Minority Oversampling Technique).
# NSL-KDD has severe class imbalance:
#   U2R: only ~52 rows (0.04% of dataset)
#   R2L: only ~995 rows (0.79% of dataset)
# Use SMOTE in nslkdd_preprocessor.py to oversample these minority classes
# before training, improving U2R and R2L recall significantly.
#
# Usage (optional — only needed if SMOTE is enabled in training):
#   from imblearn.over_sampling import SMOTE
#   sm = SMOTE(random_state=42)
#   X_resampled, y_resampled = sm.fit_resample(X_train, y_train)
